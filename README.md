# Punch Line Beta

- 펀치라인 키워드 검색 애플리케이션 구축 목표 제작 중

### 목적

- 리액트 함수형 컴포넌트로 Hooks 사용
- Expo로 Webview 적용하여 모바일 애플리케이션 빌드

### API

- 네이버 뉴스 검색 API 사용
- 네이버 API가 백엔드 서버에서 호출하는 방식으로 작동해서 Java Spring에서 API 호출부 구현

### AWS Infra

- Elastic Beanstalk (Back-end)
- S3 (Front-end)
- Route 53
- Cloud Front 적용
- https 프로토콜
